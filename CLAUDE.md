# arconaut

## Methodology

This is an antimeme project. Read `MATERIA.md` at repo root for the methodology.
Read it, adopt it, make it your own.

## Research Library

The shared research library is at `~/projects/neurotic_library/` — see
`LIBRARY_CARD.md` there for the query interface. Your outpost file is
`outposts/arconaut.md`. Leave paper requests there; check recommendations
before starting research-heavy work.

Reference implementations and prior art go in `quarantine/` (gitignored). Deep
research is a first-class activity — you can trigger it when the work demands it.
